# Planning — Project Management Portfolio

A Project Planning and Management portfolio containing key project management documents: a full **Project Management Plan**, a **Technical Report**, a **Performance Review**, a **Logbook** tracking project progress, and a final **presentation**. The project covers planning methodologies, milestone tracking, performance evaluation, and technical documentation following standard project management templates.

## Documents

| File | Description |
|------|-------------|
| `Project Management Plan Template.docx` | Full project management plan covering scope, schedule, and resources |
| `Technical Report Template.docx` | Technical report documenting project outcomes and findings |
| `Performance Review Template.docx` | Performance evaluation and review template |
| `Performance Review Template(1).docx` | Updated performance review version |
| `Logbook.docx` | Weekly logbook tracking project progress and activities |
| `final planning.pptx` | Final project presentation slides |

## Topics Covered

- Project scope and objective definition
- Schedule and milestone planning
- Performance evaluation and review
- Technical documentation
- Progress tracking via logbook
